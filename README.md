# Basic-Typing-Test

This is a simple Python program to test your typing speed and accuracy using the 1000 most common English words.

## Features:
This tool randomly generates a string of words for you to type.
It measures how fast you type in words per minute (WPM).
It calculates how accurate your typing is as a percentage.
You will need Python 3+ and the wordfreq library.

## Requirements:
Python 3+ and
[Wordfreq](https://pypi.org/project/wordfreq/)

Install ```wordfreq``` with: ```pip3 install wordfreq```

## How to Use:

1. Run the programm:
```bash
python typing_test.py
```
2. Enter how many words you want to test yourself on (max 1000):
```bash
With how many words do you want to test yourself? (max 1000):
```
3. The program will display a string of random words. Read it carefully.
4. Press ```ENTER``` when ready to start typing.
5. Type the displayed text as accurately and quickly as possible, then press ```ENTER``` when done.
6. Results will be displayed:
   
```bash
--Results--
WPM = ... , Time = ... , Accuracy = ...
```
